# Scratch 3.0 计算器使用说明

## 项目简介

这是一个基于 Scratch 3.0 的计算器程序，支持基础的四则运算（加、减、乘、除）。

## 功能特性

- ✅ 加法运算 (+)
- ✅ 减法运算 (-)
- ✅ 乘法运算 (×)
- ✅ 除法运算 (÷)
- ✅ 清除功能 (C)
- ✅ 实时显示计算结果

## 使用方法

### 1. 打开项目

1. 访问 [Scratch 官网](https://scratch.mit.edu)
2. 点击"创建"或"加入"
3. 在编辑器中选择"文件" → "从你的电脑中上传"
4. 选择 `calculator.sb3` 文件上传

### 2. 运行计算器

1. 点击绿旗 🚩 启动程序
2. 屏幕上方会显示一个大的变量监视器，显示当前计算结果

### 3. 进行计算

计算器支持通过键盘输入进行操作：

**数字输入：**
- 按键盘上的 `0-9` 输入数字

**运算符：**
- 按 `+` 进行加法运算
- 按 `-` 进行减法运算
- 按 `*` 进行乘法运算
- 按 `/` 进行除法运算

**其他功能：**
- 按 `=` 或 `Enter` 计算结果
- 按 `C` 或 `c` 清除显示，重新开始计算

### 4. 计算示例

**例1：简单加法**
```
输入: 5 + 3 =
结果: 8
```

**例2：连续运算**
```
输入: 10 - 4 =
结果: 6
再输入: * 2 =
结果: 12
```

**例3：除法运算**
```
输入: 20 / 4 =
结果: 5
```

## 项目结构

计算器项目包含以下主要组件：

### 变量
- **显示 (display)**: 当前显示的数字或结果
- **数字1 (num1)**: 存储第一个操作数
- **数字2 (num2)**: 存储第二个操作数
- **运算符 (operator)**: 存储当前选择的运算符
- **新数字 (newNumber)**: 标记是否开始输入新数字

### 主要脚本逻辑

1. **初始化**: 当绿旗被点击时，重置所有变量
2. **数字输入**: 捕获键盘数字输入，更新显示
3. **运算符处理**: 保存第一个数字和运算符
4. **计算结果**: 执行运算并显示结果
