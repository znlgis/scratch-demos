# Scratch 3.0 计算器使用说明

## 项目简介

这是一个基于 Scratch 3.0 的计算器程序，采用**按钮式界面**（类似 Windows 计算器），支持基础的四则运算（加、减、乘、除）。

## 功能特性

- ✅ 加法运算 (+)
- ✅ 减法运算 (-)
- ✅ 乘法运算 (×)
- ✅ 除法运算 (÷)
- ✅ 清除功能 (C)
- ✅ 实时显示计算结果
- ✅ **可点击按钮界面**（鼠标操作）
- ✅ Windows 计算器风格布局

## 使用方法

### 1. 打开项目

1. 访问 [Scratch 官网](https://scratch.mit.edu)
2. 点击"创建"或"加入"
3. 在编辑器中选择"文件" → "从你的电脑中上传"
4. 选择 `calculator.sb3` 文件上传

### 2. 运行计算器

1. 点击绿旗 🚩 启动程序
2. 屏幕上方会显示一个大的变量监视器，显示当前计算结果
3. 舞台上会显示所有计算器按钮

### 3. 进行计算

计算器通过**鼠标点击按钮**进行操作：

**数字按钮：**
- 点击数字按钮 `0-9` 输入数字
- 按钮排列成计算器标准布局（3×4 网格）

**运算符按钮：**
- 点击 `+` 按钮进行加法运算
- 点击 `-` 按钮进行减法运算
- 点击 `×` 按钮进行乘法运算
- 点击 `÷` 按钮进行除法运算

**其他功能按钮：**
- 点击 `=` 按钮计算结果
- 点击 `C` 按钮清除显示，重新开始计算

### 4. 计算示例

**例1：简单加法**
```
操作: 点击 [5] → [+] → [3] → [=]
结果: 8
```

**例2：连续运算**
```
操作: 点击 [1] → [0] → [-] → [4] → [=]
结果: 6
再点击: [×] → [2] → [=]
结果: 12
```

**例3：除法运算**
```
操作: 点击 [2] → [0] → [÷] → [4] → [=]
结果: 5
```

## 项目结构

计算器项目包含以下主要组件：

### 角色（Sprites）
- **Controller**: 控制器角色（不可见），负责初始化
- **Button_0 ~ Button_9**: 10个数字按钮（0-9）
- **Button_plus**: 加法按钮 (+)
- **Button_minus**: 减法按钮 (-)
- **Button_multiply**: 乘法按钮 (×)
- **Button_divide**: 除法按钮 (÷)
- **Button_equals**: 等号按钮 (=)
- **Button_clear**: 清除按钮 (C)

总计：**18 个角色**（1 个控制器 + 17 个按钮）

### 变量
- **显示 (display)**: 当前显示的数字或结果
- **数字1 (num1)**: 存储第一个操作数
- **数字2 (num2)**: 存储第二个操作数
- **运算符 (operator)**: 存储当前选择的运算符
- **新数字 (newNumber)**: 标记是否开始输入新数字

### 按钮布局

```
[C]              [7] [8] [9] [+]
                 [4] [5] [6] [-]
                 [1] [2] [3] [×]
                 [0]     [=] [÷]
```

### 主要脚本逻辑

1. **初始化** (Controller): 当绿旗被点击时，重置所有变量
2. **数字按钮**: 点击后更新显示变量
   - 如果是新数字，替换显示内容
   - 如果不是新数字，追加到显示内容
3. **运算符按钮**: 保存第一个数字和运算符，准备输入第二个数字
4. **等号按钮**: 执行计算并显示结果
5. **清除按钮**: 重置所有变量到初始状态

## 技术细节

### 项目信息
- **Scratch 版本**: 3.0
- **文件格式**: .sb3 (Scratch 3.0 项目文件)
- **文件大小**: 约 8.3 KB
- **兼容性**: 可在 Scratch 3.0 在线编辑器和桌面版中运行

### 交互方式
- **输入方式**: 鼠标点击按钮（不支持键盘输入）
- **事件类型**: `when this sprite clicked` (当角色被点击)
- **按钮总数**: 17 个可点击按钮

### 文件说明
- `calculator.sb3`: Scratch 3.0 项目文件（可直接在 Scratch 中打开）
- `CALCULATOR_README.md`: 本说明文档

### 按钮设计
- **数字按钮**: 灰色背景 (#e0e0e0)
- **运算符按钮**: 橙色背景 (#ff9800)
- **等号按钮**: 绿色背景 (#4caf50)
- **清除按钮**: 红色背景 (#f44336)
- **所有按钮**: 圆角矩形设计，带边框
