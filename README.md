# scratch-demos
Scratch 3.0 游戏演示项目 / Scratch 3.0 Game Demos

## 五子棋 (Gomoku / Five in a Row)

### 简介 / Introduction
这是一个基于 Scratch 3.0 实现的五子棋游戏。
This is a Gomoku game implemented in Scratch 3.0.

### 游戏规则 / Game Rules
- 两名玩家轮流在 15×15 的棋盘上落子
- 黑子先行，白子后行
- 首先在横、竖或斜方向上连成五子的一方获胜

- Two players take turns placing pieces on a 15×15 board
- Black moves first, then White
- The first player to get 5 pieces in a row (horizontally, vertically, or diagonally) wins

### 如何运行 / How to Play
1. 前往 [Scratch 官网](https://scratch.mit.edu/)
2. 点击"创作" (Create)
3. 选择"文件" -> "从电脑中上传" (File -> Load from your computer)
4. 选择 `gomoku.sb3` 文件
5. 点击绿旗开始游戏

1. Go to [Scratch website](https://scratch.mit.edu/)
2. Click "Create"
3. Select "File" -> "Load from your computer"
4. Choose the `gomoku.sb3` file
5. Click the green flag to start playing

### 操作说明 / Controls
- 点击棋盘上的交叉点放置棋子
- 变量 "turn" 显示当前回合 (1=黑子, 2=白子)

- Click on the board intersections to place pieces
- The "turn" variable shows the current turn (1=Black, 2=White)
